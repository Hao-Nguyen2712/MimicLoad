# MimicLoad - User Simulator Engine

**MimicLoad** is a high-performance User Simulator designed to push the boundaries of traditional load testing tools. Instead of merely generating "brute-force" load, MimicLoad focuses on simulating the randomness and complexity of human behavior through mathematical models and artificial intelligence.

## 🚀 Core Philosophy: "Smart Load > Heavy Load"

Modern security systems (WAF, Anti-bot) can easily detect and block linear traffic patterns. MimicLoad solves this by:
- **Markov Chain Logic:** Routing virtual users based on probabilistic scenarios.
- **Gaussian Think-time:** Simulating random wait times between actions.
- **Identity Spooling:** Rotating thousands of Residential Proxies and Device Fingerprints.

## 🛠 Tech Stack

Built on a **Modular Monolith** architecture to optimize development resources while ensuring high performance and scalability.

- **Backend:** .NET 10 
- **Load Engine:** NBomber
- **Frontend:** Angular 21
- **AI Orchestration:** Microsoft Semantic Kernel
